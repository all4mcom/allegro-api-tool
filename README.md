# Pobranie informacji o aukcji

Zestaw prywatnych skryptów wykorzystujących REST API Allegro do zarządzania ofertami sprzedaży.

Funkcjonalność obejmuje między innymi:

* pobieranie informacji o ofertach i ich parametrach,
* odczyt aktualnych cen oraz stanów magazynowych,
* aktualizację ilości dostępnych produktów,
* automatyczną zmianę cen ofert,
* zarządzanie zakresami cen i regułami cenowymi,
* tworzenie nowych ofert sprzedaży,
* edycję istniejących ofert,
* aktualizację opisów ofert,
* pobieranie danych niezbędnych do monitorowania i zarządzania sprzedażą.

Aplikacja jest wykorzystywana wyłącznie do obsługi własnych ofert i automatyzacji procesów sprzedażowych poprzez oficjalne REST API Allegro.

Technologia: Python, REST API Allegro.
